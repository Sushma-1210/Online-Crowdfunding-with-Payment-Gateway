This project is a fully functional online crowdfunding platform built using modern web technologies. It allows users to create and support fundraising campaigns, track progress, and securely donate through a simulated payment gateway.

✨ Features
👤 User Registration & Login

📢 Campaign Creation with details and images

💰 Secure (simulated) payment gateway integration for donations

📊 Campaign progress tracking with real-time updates

📄 Campaign listing and detailed views

📬 Contact & feedback support

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript

Backend: Flask (Python)

Database: MySQL / SQLite

Payment Gateway: Razorpay (mock/demo integration or API simulation)

📌 How It Works
Users can register and log in to create or support campaigns.

Donors can view all campaigns and contribute securely.

Admins can monitor campaign activities and manage users.
